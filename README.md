# 第一次考核

## 编程

1. 给你一个数，求最大公因数，用循环
2. 如果我只让你用一个字节的一个变量，存储8位数，请问如何如何把它提取出来这8个数，用循环
3. 给一句英文，例如`how old are you?`，将英文句子首字母大写，句子自己给自己出
4. 输入3个数（没有0），做到将其合成最大的三位整数。例如 5、9、7，输出975
5. scanf输入一个字符串的`ASCII`码，输出其转译后的==字符串==形式
